# 分析師封鎖統計網站 - 設計思路

## 選定設計方向

### 設計主題：**數據驅動的專業儀表板**

### 設計哲學
本網站採用**現代金融科技風格**，強調數據的清晰展示、快速對比和深度分析。設計目標是為投資分析師提供一個**高效、直觀、專業**的工具，讓複雜的統計數據變得易於理解。

### 核心原則
1. **數據優先**：表格和圖表是主角，所有視覺設計都為數據展示服務
2. **快速掃描**：使用顏色編碼、對比度和層級來幫助用戶快速定位關鍵信息
3. **互動深度**：從概覽到詳情的漸進式披露，支援多時間維度的對比
4. **專業感**：使用冷色調、清晰的排版和精確的間距

### 色彩哲學
- **主色**：深藍色 (`#1e40af`) - 代表信任、專業、穩定
- **輔色**：青綠色 (`#0891b2`) - 代表增長、活力
- **強調色**：紅色 (`#dc2626`) - 代表下降、警示（用於負數據）
- **背景**：淺灰藍 (`#f8fafc`) - 減少眼睛疲勞，提升閱讀舒適度
- **文字**：深灰 (`#1e293b`) - 高對比度，易於閱讀

### 版面範例
- **主頁**：頂部時間切換按鈕，下方分為大來和大華兩個分類卡片，每個卡片內是分析師列表
- **詳細頁**：標題 + 時間篩選按鈕（今年、近三月、近一月、上周、昨日），下方兩個圖表（累積好友數、每日新增與封鎖）

### 簽名元素
1. **卡片設計**：帶有細微陰影和邊框的卡片，營造層級感
2. **數字強調**：大號數字配合小號標籤，快速傳達信息
3. **漸變線條**：圖表使用漸變色，增加視覺吸引力
4. **圓角按鈕**：統一的圓角設計，現代感十足

### 交互哲學
- **按鈕狀態**：選中時填充顏色，未選中時為淺色邊框
- **卡片點擊**：點擊分析師卡片時，平滑過渡到詳細頁
- **數字變化**：時間切換時，表格數據平滑更新（可選動畫）

### 動畫指南
- **按鈕點擊**：100-150ms 的快速反饋，使用 `scale(0.97)` 效果
- **頁面過渡**：200-300ms 的淡入淡出效果
- **圖表更新**：300-500ms 的平滑動畫，使用 easing-out
- **列表進入**：每項延遲 30-50ms，創造瀑布效果

### 排版系統
- **標題**：`font-bold text-xl` - 深藍色，用於頁面標題
- **副標題**：`font-semibold text-base` - 深灰色，用於分類標題
- **正文**：`font-normal text-sm` - 深灰色，用於數據和說明文本
- **數字**：`font-mono font-bold text-lg` - 深藍色，用於重要數據

### 品牌精髓
**一句定位**：為投資分析師提供實時、精確、易用的 LINE/YT 封鎖統計工具，幫助快速了解粉絲動態。

**品牌個性**：專業、可靠、高效

### 品牌聲音
- **標題示例**：「本周增長排行」、「分析師績效對比」
- **CTA 示例**：「查看詳細趨勢」、「切換時間維度」
- **禁用表達**：避免使用「歡迎」、「立即開始」等通用詞彙

### Logo 設計
使用簡潔的圖表符號：一個上升的折線圖，配合深藍色，代表數據增長和分析。

### 簽名品牌色
**深藍色** (`#1e40af`) - 代表專業、信任、穩定

## 實現細節

### 頁面結構
1. **主頁** (`/`)
   - 頂部：時間切換按鈕（本周增長、本月增長、今年增長）
   - 中部：大來國際投顧分析師列表
   - 下部：大華國際投顧分析師列表
   - 每個分析師卡片顯示：名稱、日期、選中時間的增長數、封鎖率、本周封鎖

2. **詳細頁** (`/analyst/:name`)
   - 頂部：分析師名稱 + 返回按鈕
   - 時間篩選按鈕：今年、近三月、近一月、上周、昨日
   - 圖表 1：累積好友數趨勢
   - 圖表 2：每日新增與封鎖趨勢

### 顏色定義
```css
--primary: #1e40af (深藍)
--secondary: #0891b2 (青綠)
--destructive: #dc2626 (紅)
--background: #f8fafc (淺灰藍)
--foreground: #1e293b (深灰)
--border: #e2e8f0 (淺灰)
```

### 字體
- 標題：`font-bold`
- 副標題：`font-semibold`
- 正文：`font-normal`
- 數字：`font-mono font-bold`
